<vbox> <label style:font='"Roboto Monospace"' style:fontsize='20'>value</label> <hbox> <label id='myelement' style:font='"Roboto Monospace"' style:fontsize='20'> &lt;b&gt;value2&lt;/b&gt; </label> <button onclick='{ print "button pressed". }'></button> </hbox> </vbox>
This tool creates Kerboscript from XML files, crafted to represent GUI-Layouts.


If you want to set a labels font, you say:
  <label style:font='"Roboto Monospace"'></label>
Note that both single and doublequotes are used to represent a string value!

This way it is possible to add inline anonymous functions as well:
  <button onclick='{ print "button pressed". }'></button>

If you want to use Rich Text, you need to HTML-Encode the text:
  &lt;b&gt;value2&lt;/b&gt;


Adding the generated code into your project is easy:
  Save the generated code to a ".ks"-file, lets call it "myinterface.ks".
  Call the file from your code and pass it a GUI to render to:
    local g is GUI(400).
    run "myinterface.ks"(g).


Using ID:
  IDs can be use to dynamically pick certain elements from your interface.
  First add an ID-Attribute to the element you want to use inside your own code:
    <label id='mylabel'></label>
  Now you can pass a second argument when running the interfaces script:
    local g is GUI(400).
    local ids is LEX().
    run "myinterface.ks"(g, ids).
    set ids["mylabel"]:TEXT to "My pretty text.".
